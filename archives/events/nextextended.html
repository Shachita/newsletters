Cloud Next Extended '19

Event Description

Google Cloud Next'19 was a 3-day event held at San Francisco where a lot of developers, influencers, customers, engineers, learneds and the larger global cloud community came together to unlock new opportunities and get inspired to learn about Google Cloud Platform and various intriguing cloud- related technologies.

Google Cloud Next Extended'19, was an event organised by GCDC, Indore, where the team reviewed the diverse cloud technologies introduced at the Google Cloud Next'19 event at San Francisco. Some wonderful speakers shared their experience of attending cloud next and conducted sessions on several topics like Anthos, kubernetes, Microsoft flow, Cloud Run,etc. This was a golden opportunity for those who could not be a part of Cloud Next'19.

Anthos

Dharmesh Vaya

Key-points

  • Anthos is an integrated platform that helps you modernize how you develop, secure, and operate hybrid cloud and cloud-native environments.
  • Work on the principle of “Write Once, Run Anywhere”.
  • GKE, GKE On-Prem, and Anthos Config Management are the core building blocks of Anthos.
  • Anthos can be used in conjunction with the open-source Istio offering.
  • Anthos Migrate is free for customers to use when migrating workloads to GCP.

Kubernetes for Machine Learning

Akash Agrawal

Key-points

  • Kubernetes is a portable, extensible open-source platform for managing containerized workloads and services, that facilitates both declarative configuration and automation.
  • Kubernetes provides a container-centric management environment. It orchestrates computing, networking, and storage infrastructure on behalf of user workloads.
  • Kubeflow are focused on development of Machine Learning workflow on Kubernetes(Simple, Portable, Scalable).
  • Kubeflow is to support deployment of best-of-breed open source system for ML to diverse infrastructure.

Microsoft Flow

Key-points

  • Microsoft Flow automates repetitive tasks, allowing you to devote more of your time to important matters.

Cloud Run

Tushar Pal

Key-points

  • Cloud Run is a managed compute platform that enables you to run stateless containers that are invocable via HTTP requests.
  • Cloud Run built from Knative, letting you choose to run your containers either fully managed with Cloud Run, or in your Google Kubernetes Engine cluster with Cloud Run on GKE.
  • Cloud Run enables you to run stateless HTTP workloads without having to worry about servers.
  • Deploy your stateless containers with a consistent developer experience to a fully managed environment or to your own GKE cluster.
  • Use the programming language of your choice, any language or operating system libraries, or even bring your own binaries.
