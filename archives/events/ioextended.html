I/O Extended '19

Event Description

Google IO 2019 was a 3-day event held at Mountain View, CA where a lot of developers, influencers, customers, engineers, learneds and the larger global google community came together to unlock new opportunities and get inspired to learn about the latest inventions and trends in google technologies.

Google IO Extended 2019 was an event organised by GDG Indore in collaboration with GDG Cloud Indore and WTM Indore where the team reviewed the diverse technologies introduced at the Google IO Extended 2019 event at Mountain View, CA. In the event we had a discussion and hands of practice over Machine Learning, Tensorflow, and Kotlin succeeded by the 18 Hour long Hackathon on Kotlin.

TensorFlow

Sandeep Gupta

Key-points

  • TensorFlow is an end-to-end open source platform for machine learning.
  • It has a comprehensive, flexible ecosystem of tools, libraries and community resources that lets researchers push the state-of-the-art in ML and developers easily build and deploy ML powered applications.
  • Tensorflow is easy (Simplified APIs), powerful (flexibility and performance) and scalable (Deploy everywhere).
  • Tensorflow's name is directly derived from its core framework: Tensor. In Tensorflow, all the computations involve tensors which are vectors or matrices of n-dimensions that represents all types of data.
  • The accessibility of Tensorflow for each and every one and it’s incorporation of APIs to make a scalable deep learning architecture bring it into popular demand.

Kotlin for Java Devs

Pradhan Yash Sharma

Key-points

  • Kotlin is a general purpose, open source, statically typed “pragmatic” programming language for the JVM and Android that combines object-oriented and functional programming features.
  • A better Java for the JVM which explicitly provides null safety and supports immutability. Reduction of boiler plate code is the huge triumph of Kotlin for all the Android developers.
  • Gradually shift from the language you code in to Kotlin with the help of enormous features provided in Kotlin.
  • Fortunately, if you are new to language and don’t know how to convert your Java code into Kotlin, Java to Kotlin convertor is at your rescue.
