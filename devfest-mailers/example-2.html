What would you say if we tell you that you can learn building a fast & reliable website, a mobile app, host applications on the cloud & analyse data, build an end-to-end predictive machine learning model and work on a cross platform application using Flutter - just in 3 days?

50 Google Developer Groups in India have come together for the first time to organize one of the biggest developer events of 2020 - DevFest India 2020 from 16-18 October, 2020 (4-9 PM IST daily). We will have experts on 5 tracks take you through projects which can make you a better developer by adding new skills and capabilities in your toolbox.

Whether you’re a beginner or an experienced developer, there is ample content planned for you.
  • In the Mobile track, you’ll witness how to build a clone of the BookMyShow Application using Kotlin as a programming language.
  • In the Machine Learning track, there will be demonstration on building end to end prediction pipelines using ML, managing CI/CD pipelines for ML algorithms and a case study on detecting tumours in health care reports.
  • In the Cloud Track, the experts will guide you with the first steps to get started with Google Cloud, how to deploy your applications on Google Cloud and working on large datasets using BigQuery.
  • In the Web track, the experts will demonstrate their code on how to build a Pokemon website using Angular and later convert it to PWA. Experts will also share tips on SEO and improving the performance of your website.
  • In the Cross Platform Development track, experts will showcase their code on how to build applications that can be deployed as mobile apps and as well as run as web applications using Flutter.
Join us at DevFest India 2020 to learn new skills, connect with other developers and experts from different domains across the country and have a fun filled weekend. You can also win exciting prizes during the 3 day event.

See you all at DevFest India 2020!
